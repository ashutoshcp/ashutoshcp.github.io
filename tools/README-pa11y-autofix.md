Pa11y Auto-fix helper

This helper script applies conservative fixes to HTML files based on a pa11y JSON report.

Requirements:
- Python 3
- BeautifulSoup4: pip3 install --user beautifulsoup4

Workflow:
1. Run a pa11y JSON report locally (pa11y must be installed via npm):
   npx pa11y http://localhost:8000/ --reporter json > pa11y-report.json

2. Run the autofix tool from the repo root:
   python3 tools/pa11y_autofix.py --report pa11y-report.json --root .

What the tool does (conservative):
- Adds alt="" to <img> elements missing alt attributes.
- Adds aria-label to empty <a> links containing only icons (derives label from the href).

Important:
- The tool creates a backup of any edited file at <file>.pa11y-autofix.bak. Review changes and replace autogenerated alt="" with meaningful text where possible.
- This is not a replacement for manual accessibility review.
